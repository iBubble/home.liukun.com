### 技术栈
- **前端框架**: React 18 + TypeScript
- **构建工具**: Webpack 5
- **样式**: Tailwind CSS
- **路由**: React Router 6
- **状态管理**: Zustand
- **动画**: Framer Motion
- **国际化**: i18next + react-i18next
- **进程管理**: PM2
- **Web 服务器**: Nginx

### 核心功能
1. **双模式主题系统**（梦境/现实）
2. **老虎机入口**（10% 梦境，90% 现实）
3. **故障艺术效果**（雪花点、色彩分离、画面扭曲）
4. **光标轨迹效果**（硬币/针头图标串）
5. **对比滑块**（梦境/现实场景对比）
6. **角色卡牌**（RPG 风格展示）
7. **多语言支持**（中英意三语）
8. **响应式设计**（桌面/平板/移动端）

### 页面内容
1. **首页**: 老虎机入口，欢迎文案
2. **故事板**: 梦境/现实场景对比，反讽文案
3. **角色**: 3个主要角色（李明、小美、老王）
4. **导演风格**: 导演简介、获奖记录、创作理念、参考影片
5. **众筹**: 项目进度、回报档位、支持表单、FAQ
6. **投资人专区**: 市场分析、财务规划、核心团队、发行策略
