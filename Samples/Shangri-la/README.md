# 天空之境·数智香格里拉 - 迪庆智慧文旅低空经济方案

## 项目简介

这是一个展示迪庆州"低空经济+智慧文旅"空地一体化建设方案的交互式网站。网站采用纯前端技术开发，无需后端服务器，可直接在浏览器中运行。

## 访问地址

**线上访问：** http://ibubble.vicp.net:81/Sample/

## 项目特点

- ✅ 纯前端开发，无需构建工具
- ✅ 响应式设计，支持移动端和桌面端
- ✅ 丰富的数据可视化图表
- ✅ 平滑滚动和动画效果
- ✅ 香格里拉地域特色设计
- ✅ **Unsplash高质量图片** - 稳定可靠的图片源
- ✅ **16张精选图片** - 涵盖景点、应用场景
- ✅ **全球CDN加速** - 快速加载，无访问限制
- ✅ 图片悬停特效和过渡动画

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **JavaScript (ES6+)** - 交互逻辑
- **Chart.js** - 数据可视化
- **D3.js** - 高级可视化（预留）

## 本地运行

### 方法1：使用启动脚本（推荐）
```bash
bash start-server.sh
```
然后在浏览器中访问 `http://localhost:8000`

### 方法2：直接打开文件
1. 直接在浏览器中打开 `index.html` 文件

### 方法3：使用Python服务器
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

### 方法4：使用PHP服务器
```bash
php -S localhost:8000
```

访问 `http://localhost:8000`

## 部署说明

### 快速部署（使用脚本）

```bash
bash deploy.sh
```

脚本会自动将所有必要文件上传到服务器。

### 手动部署

1. 将以下文件上传到服务器目录：
   - index.html
   - styles.css
   - script.js

2. 确保文件权限正确：
   ```bash
   chmod 644 index.html styles.css script.js
   ```

3. 访问对应URL即可

### 服务器要求

- 支持静态文件托管
- 无需PHP、Node.js等后端环境
- 建议启用Gzip压缩
- 建议配置CDN加速

### CDN依赖

网站使用以下CDN资源：
- Chart.js 4.4.0
- D3.js v7
- Font Awesome 6.4.0
