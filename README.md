## 🚀 快速开始

### 本地运行

1. 克隆仓库：
```bash
git clone https://github.com/ibubble/home.liukun.com.git
cd home.liukun.com
```

2. 使用任意 Web 服务器运行：
```bash
# 使用 Python
python -m http.server 8000

# 或使用 PHP
php -S localhost:8000

# 或使用 Node.js (需要安装 http-server)
npx http-server
```

3. 在浏览器中访问 `http://localhost:8000`

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript (原生)
- **后端**: PHP 8.2
- **服务器**: Nginx + 宝塔面板
- **样式**: 自定义 CSS (赛博朋克主题)
